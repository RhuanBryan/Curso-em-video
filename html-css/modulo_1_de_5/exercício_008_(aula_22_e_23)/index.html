Principais formataçoes

Negrito

Dessa forma temos um termo em negrito usando a tag <b> (nao semantica)

Nessa frase tenho um termo em destaque usando <strong> (semantica)

Italico

Nessa frase temos um termo em italico usando a tag <i> (nao semantica)

Nesta frase temos um texto em enfase usando a tag <em> (semantica)

Texto marcado

Podemos criar tambem um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragrafo

Texto excluido/deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas nao considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma enfase para indicar quer ele foi adicioando depois (semantica)

Existe tambem o sublinhado com a tag <U> (nao semantica)

Texto sobrescrito

Para inserir coisas do tipo: x20+3

Texto subscrito

Para inserir coisas do tipo: H2O
