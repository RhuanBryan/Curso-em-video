Principais formatações

Negrito

Dessa forma temos um termo em negrito usando a tag <b> (não semântica)

Nessa frase tenho um termo em destaque usando <strong> (semântica)

Italico

Nessa frase temos um termo em italico usando a tag <i> (não semântica)

Nesta frase temos um texto em enfase usando a tag <em> (semântica)

Texto marcado

Podemos criar tambem um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragráfo

Texto excluído/deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase para indicar quer ele foi adicioando depois (semântica)

Existe também o sublinhado com a tag <U> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo: x20+3

Texto subscrito

Para inserir coisas do tipo: H2O
